# 📊 Previsão de Violação de SLA em Telecom

## 🚀 Descrição
Este projeto tem como objetivo prever se um incidente de telecom irá violar o SLA (Service Level Agreement), utilizando técnicas de Machine Learning.

## 🎯 Problema de Negócio
Incidentes que violam SLA geram:
- Multas contratuais
- Insatisfação do cliente
- Impacto operacional

## 🧠 Solução
Foi desenvolvido um modelo de classificação utilizando Random Forest para prever a violação de SLA.

## 📊 Resultados
- Accuracy: 73%
- AUC: 0.76
- Melhor recall para classe de violação (True): 0.88

## 🔍 Principais Features
- Prioridade do incidente
- Impacto ao cliente
- Custo de material
- Quantidade de materiais

## 🛠 Tecnologias
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## 📁 Estrutura
- `etl_telecom_gcp_powerbi.ipynb` → Notebook com pipeline completo

## 🔗 Objetivo
Antecipar falhas operacionais e apoiar a tomada de decisão em telecom.
